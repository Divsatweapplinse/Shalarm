# Shalarm

一个闹钟应用，闹铃响时，需要用力摇晃手机达到指定力度，才能停止闹铃，杜绝回笼觉。为了让摇晃力度能够匹配自身力量，建议先在应用的设置中对摇晃力度进行校准。
